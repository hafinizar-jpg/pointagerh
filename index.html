Securite

Concu pour eviter la fraude

Plusieurs niveaux de securite pour garantir des donnees de pointage fiables et incontestables.

🔑
Code PIN unique
Chaque salarie possede un code a 4 chiffres personnel. Impossible de pointer a la place d'un collegue.
📷
Photo horodatee
A chaque pointage, une photo est prise automatiquement avec date, heure et nom du site en incrustation.
📍
GPS anti-triche
Le pointage est refuse si le salarie est en dehors du perimetre autorise du site (rayon configurable).
⏱
Anti-doublon
Impossible de pointer deux fois de suite en moins de 30 secondes. Evite les erreurs de manipulation.
🚨
Detection automatique
Si un salarie ne pointe pas dans les 5 minutes apres son heure prevue, il passe automatiquement en Absent.
🔒
Mot de passe admin
Les onglets sensibles (salaries, paie, absences) sont proteges par un mot de passe administrateur.

FAQ

Questions frequentes

Tout ce que vous devez savoir sur PRO TIME.

Comment acceder au logiciel ? ▼
Cliquez simplement sur votre depot dans la liste ci-dessus. Le logiciel s'ouvre directement dans Chrome, sans aucune installation. Vous pouvez aussi l'installer sur votre bureau via les 3 points de Chrome > Installer PRO TIME.
Le logiciel fonctionne-t-il sans Internet ? ▼
En mode local (sans Internet), les donnees sont sauvegardees sur l'ordinateur. Des que la connexion revient, cliquez Synchroniser et les donnees remontent sur le Reporting DG automatiquement.
Comment ajouter un salarie ? ▼
Ouvrez votre depot > onglet Salaries > + Ajouter. Renseignez nom, poste, departement, photo, code PIN et planning hebdomadaire. Le salarie peut pointer immediatement.
Comment exporter les donnees pour la comptabilite ? ▼
Onglet Fiche de paie > selectionnez le mois > cliquez Export CSV comptabilite. Le fichier Excel contient toutes les heures, retards et absences par salarie, pret a envoyer a votre service comptabilite.
Que faire si un salarie oublie son code PIN ? ▼
Allez dans l'onglet Salaries > cliquez Fiche sur la carte du salarie > modifiez le code PIN > cliquez Enregistrer. Communiquez le nouveau code au salarie en prive.
Les donnees sont-elles securisees ? ▼
Oui. Les donnees sont stockees sur Firebase (serveurs Google) avec chiffrement. Seuls les responsables autorises y ont acces. Les photos de pointage sont stockees localement sur l'ordinateur du depot.
